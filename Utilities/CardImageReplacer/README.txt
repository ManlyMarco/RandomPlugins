Replaces main preview image in cards for Illusion games that use PNG files for character cards.

How to use
- Start the app
- Select your card
- Select the new image you want to use
- Select where to save your card with its image replaced

Alternatively you can run the app through console:
CardImageReplacer.exe [Card path] [Repalcement image path] [Save path]
